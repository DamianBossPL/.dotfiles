# DamianBossPL's .dotfiles

This repository contains all of the **dotfiles** that I use for my Linux setup.

| Component             | Tool / Setup                  |
| --------------------- | ----------------------------- |
| Window Manager        | Sway                          |
| Status Bar            | Waybar                        |
| Menus                 | Rofi                          |
| Notification Daemon   | SwayNC                        |
| Shell                 | Fish                          |
| Core Packages & Tools | See below / Required Packages |

## Install

### Arch Linux - yay

```sh
yay -S ttf-firacode-nerd ttf-ms-win11-auto ttf-twemoji-color bluez bluez-obex bluez-utils blueman fish git openssh stow yay cliphist gvfs mako network-manager-applet pavucontrol power-profiles-daemon python-pywal rofi rofi-emoji sway thunar thunar-volman waybar wl-clip-persist xdg-desktop-portal xdg-desktop-portal-gtk xdg-desktop-portal-hyprland
```
