# DamianBossPL's .dotfiles

This repository contains all of the **dotfiles** that I use for my Linux setup.

| Component             | Tool / Setup                  |
| --------------------- | ----------------------------- |
| Window Manager        | Sway                          |
| Status Bar            | Waybar                        |
| Menus                 | Rofi                          |
| Notification Daemon   | Mako                          |
| Shell                 | Zsh + Oh My Zsh               |
| Core Packages & Tools | See below / Required Packages |

## Install

### Arch Linux - yay

```sh
yay -S ttf-fira-code otf-font-awesome ttf-ms-win11-auto ttf-twemoji-color bluez bluez-obex bluez-utils blueman git openssh stow yay zsh zsh-autosuggestions zsh-syntax-highlighting cava cliphist gvfs mako network-manager-applet pavucontrol power-profiles-daemon python-pywal rofi rofi-emoji sway thunar thunar-volman waybar wl-clip-persist xdg-desktop-portal xdg-desktop-portal-gtk xdg-desktop-portal-hyprland
```
